创建 Telegram 机器人

为您的 Telegram 机器人创建频道：

  1. 开始与 @BotFather 聊天
  2. 输入 /start 开始会话。
  3. 输入 /newbot 创建新机器人。
  4. 为您的机器人频道命名，例如 cool_cam_bot。
  5. 为您的机器人设置用户名，例如 CoolCamBot。
  6. 复制 BotFather 分配给您新机器人的令牌，并将其粘贴到表单中。
